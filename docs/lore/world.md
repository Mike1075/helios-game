# Helios 世界观背景（简版）

## 信念系统底层逻辑说明
本信念系统档案基于 Helios 项目的设计理念与元宇宙游戏的交互目标编写，目的是让 NPC 拥有持续一致的性格逻辑与可预测但非固定的行为模式，实现高度沉浸感与长期可玩性。

### 核心设计理念
- **三层意识模型**：核心驱动（本我）、长期信念（自我）、短期目标（超我）。
- **信念 → 行为 → 行动链**：每个信念必须能推导出具体行为。
- **世界观 / 自我观 / 价值观**三分法：保证外部逻辑与内在逻辑统一。
- **事件契约机制**：NPC 可根据信念触发或响应特定事件。
